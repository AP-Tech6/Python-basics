def find_single_number(nums):
    result = 0
    for num in nums:
        result ^= num  # XOR operation
    return result

program to find largest elemnt in the array in python
the brute force approach is to sort the array and return the last element 
but it will take O(n log n) time complexity
we can also use a single pass approach to find the largest element in O(n) time complexity

def find_largest_number(nums):
    if not nums:
        return None
    largest = nums[0]
    for num in nums[1:]:
        if num > largest:
            largest = num
    return largest
