Stack in Python
Definition
A Stack is a linear data structure that follows the LIFO (Last In First Out) principle.

Meaning:
Last element inserted
↓
First element removed
Real-Life Example
Stack of plates
The plate placed last on top is removed first.
Uses of Stack
Function calls and recursion
Undo operation in editors
Browser back button
Expression evaluation
Backtracking algorithms

LIFO Principle
Push 10
Push 20
Push 30
Stack:

30  ← Top
20
10
If we remove an element:

Pop()
Output:
30
Because it was inserted last.

Stack Operations
1. Push
Adds an element to the top of the stack.

Syntax
stack.append(value)
Example
stack = []

stack.append(10)
stack.append(20)
stack.append(30)

print(stack)
Output:
[10, 20, 30]
2. Pop
Removes the top element from the stack.

Syntax
stack.pop()
Example
stack = [10, 20, 30]

x = stack.pop()

print(x)
print(stack)
Output:

30
[10, 20]
3. Peek / Top
Returns the top element without removing it.

Syntax
stack[-1]
Example
stack = [10, 20, 30]

print(stack[-1])
Output:

30
4. Check Empty
Syntax
len(stack) == 0
Example
stack = []

print(len(stack) == 0)
Output:True


Complete Example
stack = []
# Push
stack.append(10)
stack.append(20)
stack.append(30)
print("Stack:", stack)
# Peek
print("Top:", stack[-1])
# Pop
print("Removed:", stack.pop())
print("Stack after pop:", stack)
Output:
Stack: [10, 20, 30]
Top: 30
Removed: 30
Stack after pop: [10, 20,30]


Queue in Python
Definition
A Queue is a linear data structure that follows the FIFO (First In First Out) principle.

Meaning:
First element inserted
↓
First element removed
Real-Life Example
People standing in a queue
The person who comes first leaves first.

Uses of Queue
CPU Scheduling
Printer Queue
Ticket Booking System
BFS (Breadth First Search)
Task Scheduling

FIFO Principle
Insert 10
Insert 20
Insert 30
Queue:
Front → 10 20 30 ← Rear
If we remove an element:
Dequeue()
Output:10
Because it was inserted first.
Queue Operations
1. Enqueue
Adds an element at the rear of the queue.
Syntax
queue.append(value)
Example
queue = []
queue.append(10)
queue.append(20)
queue.append(30)
print(queue)
Output:[10, 20, 30]
2. Dequeue
Removes an element from the front of the queue.
Syntax
queue.pop(0)
Example
queue = [10, 20, 30]
x = queue.pop(0)
print(x)
print(queue)
Output:

10
[20, 30]
3. Front Element
Returns the first element without removing it.

Syntax
queue[0]
Example
queue = [10, 20, 30]
print(queue[0])
Output:

10
4. Rear Element
Returns the last element.

Syntax
queue[-1]
Example
queue = [10, 20, 30]
print(queue[-1])
Output:

30
5. Check Empty
Syntax
len(queue) == 0
Example
queue = []
print(len(queue) == 0)
Output:
True


Complete Example
queue = []
# Enqueue
queue.append(10)
queue.append(20)
queue.append(30)
print("Queue:", queue)
# Front
print("Front:", queue[0])
# Dequeue
print("Removed:", queue.pop(0))
print("Queue after dequeue:", queue)
Output:Queue: [10, 20, 30]
Front: 10
Removed: 10
Queue after dequeue: [20, 30]



Deque (Double Ended Queue)
Definition
A Deque (Double Ended Queue) is a linear data structure 
in which insertion and deletion can be performed from both ends

Creating a Deque
from collections import deque
dq = deque()
Deque Operations
1. Insert at Rear
Syntax
dq.append(value)
Example
from collections import deque

dq = deque()

dq.append(10)
dq.append(20)

print(dq)
Output:

deque([10, 20])
2. Insert at Front
Syntax
dq.appendleft(value)
Example
dq.appendleft(5)

print(dq)
Output:

deque([5, 10, 20])
3. Delete from Rear
Syntax
dq.pop()
Example
x = dq.pop()

print(x)
Output:

20
4. Delete from Front
Syntax
dq.popleft()
Example
x = dq.popleft()
print(x)
Output:5
5. Front Element
Syntax
    dq[0]
    Example
    print(dq[0])
6. Rear Element
    Syntax
        dq[-1]
        Example
        print(dq[-1])
Complete Example
from collections import deque
dq = deque()
dq.append(10)
dq.append(20)
dq.appendleft(5)
print(dq)
print("Front:", dq[0])
print("Rear:", dq[-1])
dq.popleft()
dq.pop()
print(dq)
Output:
deque([5, 10, 20])
Front: 5
Rear: 20
deque([10])

Monotonic Stack
A Monotonic Stack is a stack whose elements are always maintained in a specific order.

Types:
Monotonically Increasing Stack
Elements in stack are in increasing order.

Example:

[1, 3, 5, 8]
Monotonically Decreasing Stack

Elements in stack are in decreasing order.

Example:[8, 5, 3, 1]
